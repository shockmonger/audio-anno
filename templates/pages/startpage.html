Virtual Lab for Hindustani Music

Is this for you?

Description

A lot of us have been wanting to learn more about the music we would like to hear. For some, it just was that we couldn't spend enough time practising. Some of us didn't have the right instruments, others didn't have teachers that helped describe how to practise and so on.

Result? It's just challenging now to dive into music without knowing notes for sure, and all the nuances. It seems like you could always read a book - but it doesn't have the same give! Examples from books are hard to understand, as we can't hear them.

We are going about solving this problem. With multimedia interfaces to lead your way through musical skills, we present several small exercises and drills for you to exactly start working from whatever your level is to upwards.

About the Project

This is the first release for VLabs! We have several upcoming experiments and features Virtual Labs for music aims to serve the needs for experimenting with Indian music. So far we are focusing on North Indian/ Hindustani Music. This lab will have experiments, and repositories in which musical material will be accessible to all to read. This is the first release of the lab. If you use this software, please feel free to report back with feedback so we can improve the product!
